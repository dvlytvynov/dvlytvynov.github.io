Импорт с помощью командной строки:

1. Скачать архив https://github.com/datacharmer/test_db

2. Распаковать архив test_db-master.zip

3. Прописать переменную окружения MYSQL_HOME = C:\Program Files\MySQL\MySQL Server 8.0 (папка с установленной MySQL)

4. Добавить в переменную окружения Path: %MYSQL_HOME%\bin;

5. Открыть командную строку в папке test_db-master, куда распакован архив, ввести:

mysql -uroot -p < employees.sql

Enter password: **** // ввести пароль
