Создание одного коммита из нескольких


1. Находимся на старой ветке branch1 со монжеством коммитов

2. git branch <name> - создаем новую ветку на всякий случай, чтобы все не испортить name=temp-2022-06-10-22-00

3. перключаемся на main

4. с помощью gitk копируем хэш коммита1 с нужными файлами, сюда хотим переместить main

5. git reset --hard <хэш коммита1> - после этой команды текущая ветка (main) находится на коммите1, файловая система соответствует коммиту1

6. с помощью gitk копируем хэш коммита2, от этого коммита ответвится конечный нужный коммит

7. git reset --soft <хэш коммита2> - после этой команды текущая ветка (main) находится на коммите2, файловая система не изменяется, т.е. соответствует коммиту1

8. git commit

9. git branch <имя новой ветки>

10. git checkout <имя новой ветки>

9. git branch -d branch1
