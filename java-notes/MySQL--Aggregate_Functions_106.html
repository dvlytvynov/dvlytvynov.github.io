Aggregate Functions


Агрегирующие функции

SELECT COUNT(*) FROM film; // кол-во записей, которые хранятся в таблице film

SELECT MIN(rental_rate) FROM film; // максимальная цена аренды фильма

SELECT MAX(rental_rate) FROM film; // минимальная цена аренды фильма

SELECT AVG(rental_rate) FROM film; // средняя цена аренды фильма

Совмещаем все эти функции в одном запросе:

SELECT MAX(rental_rate), MIN(rental_rate), AVG(rental_rate) FROM film;

SELECT MAX(rental_rate) AS max, MIN(rental_rate) AS min, AVG(rental_rate) AS avg FROM film; // техника алеасов - задаем собственные имена колонкам

SELECT COUNT(*) FROM film; // кол-во всех записей, которые хранятся в таблице film

SELECT COUNT(original_language_id) FROM film; // считает только те строки, где в колонке original_language_id не NULL
