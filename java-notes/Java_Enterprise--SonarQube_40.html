SonarQube


SonarQube - Средство автоматизации для быстрой проверки качества кода (самый распространенный и бесплатный)

- https://www.sonarqube.org/

- Download/Community edition

- Распаковать ZIP-архив

- .\sonarqube-9.5.0.56709\sonarqube-9.5.0.56709\bin\windows-x86-64

- Открыть в этой папке командную строку (ввести cmd в строке адреса) (Если будут какая-то ошибка, то терминал не закроется, и эту ошибку можно увидеть)

→ StartSonar.bat

⇒ Должно вывести в конце (в папку .\sonarqube-9.5.0.56709\sonarqube-9.5.0.56709\temp помещаются нужные файлы)

• jvm 1 | 2022.08.07 17:44:18 INFO app[][o.s.a.SchedulerImpl] Process[ce] is up

• jvm 1 | 2022.08.07 17:44:18 INFO app[][o.s.a.SchedulerImpl] SonarQube is operational

→ В браузере: http://localhost:9000 (9000 - порт по умолчанию)

→ Запустить из терминала IDEA в своем проекте: mvn org.sonarsource.scanner.maven:sonar-maven-plugin:3.4.0.905:sonar -D sonar.login=admin -D sonar.password=admin - сканирует проект, подключается к запущенному sonar и помещает туда отчет; нужно обновить в браузере SonarQube, зайти в проект, там можно посмотреть отчет:

⇒ http://localhost:9000/projects

→ См. логи: .\sonarqube-9.5.0.56709\sonarqube-9.5.0.56709\logs

SonarQube можно прописать в плагинах в pom-файле
