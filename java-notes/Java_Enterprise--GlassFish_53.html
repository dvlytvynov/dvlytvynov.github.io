GlassFish


GlassFish - Application Server (сервер приложений):

- имеет внутри функционал контейнера сервлетов (типа Tomcat); Tomcat более популярен, чем GlassFish

- у GlassFish лучше производительность, чем у Tomcat

- может внутри содержать дополнительные базы данных, настройки к базам данных

- настройки к очередям сообщиний типа JMS (может быть внутренний JMS) (аналог - Kafka)

- намного богаче админка

- можно запустить несколько экземпляров GlassFish под разные домены (сайты), не связанные друг с другом; см. папку \glassfish6\glassfish\domains

- можно использовать JPA

- GlassFish называет сервисы (свои бины, которые содержат бизнес логику) EJB (Enerprise JavaBeans)

- работает с файлами .war

- перед изучением Spring рекомендуется изучить GlassFish

Запуск GlassFish из командной строки из папки \glassfish6\bin:

asadmin start-domain

./asadmin start-domain // стр. 25 Java EE 7 и сервер приложений GlassFish 4

Адрес админки:

localhost:4848

Меню админки:

- Clusters - у Application Server зашит в функционал работа в кластерном режиме (когда работает несколько серверов как одно целое; если один сервер падает, то остальные принимают на себя нагрузку и синхронизируют между собой

- JDBC - управление базами данных (можно прописать коннекшен к Postgres, MySQL и др. и нам не нужно будет знать в нашем приложении ничего кроме названия которое мы эдесь пропишем (JDBC Resources)

→ DerbyPool - встроенная в GlassFish база данных

- JMS Resources/Connection Factories/Edit JMS Connection Factory - управление топиками (типа Kafka)

- Configuration/server-config/Security/Realms

→ admin-realm - стандарнтный Security с session id

→ certificate

→ file - прописать файл с логинами, паролями

- Configuration/server-config/HTTP Service/Http Listeners

→ http-listener-1 - работает по http (если важна быстрота работы по сети и 100% доверяем работе по сети наших двух приложений)

→ http-listener-1 - работает по https

Пример:


C:\war>asadmin start-domain domain1
C:\war>asadmin deploy Lesson-13-micro1.war
C:\war>asadmin deploy Lesson-13-micro2.war
C:\war>asadmin list-applications
C:\war>asadmin undeploy Lesson-13-micro1
C:\war>asadmin undeploy Lesson-13-micro2
C:\war>asadmin stop-domain domain1

EAR файл - Enterprise Archive более сложный архив который в себе содержит (кроме jar, классов, htmp, css) настройки GlassFish (подключения к базе данных и т.д.)

batch - параллельная обработка больших объемов данных.
