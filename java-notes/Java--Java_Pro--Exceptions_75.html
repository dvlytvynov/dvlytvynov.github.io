Exceptions


Введение в обработку исключений

try{

int[] numbers = new int[3];

numbers[4]=45;

System.out.println(numbers[4]);

}

catch(Exception ex){

ex.printStackTrace();

}

finally{

System.out.println("Блок finally");

}

System.out.println("Программа завершена");

Выражение catch имеет следующий синтаксис: catch (тип_исключения имя_переменной). Блок finally необязательный, и его можно при обработке исключений опускать. Блок finally выполняется в любом случае, возникло ли исключение в блоке try или нет

Оператор throw

Чтобы сообщить о выполнении исключительных ситуаций в программе, можно использовать оператор throw. То есть с помощью этого оператора мы сами можем создать исключение и вызвать его в процессе выполнения. Например, в нашей программе происходит ввод числа, и мы хотим, чтобы, если число больше 30, то возникало исключение:

try{

Scanner in = new Scanner(System.in);

int x = in.nextInt();

if(x>=30){

throw new Exception("Число х должно быть меньше 30");

}

}

catch(Exception ex){

System.out.println(ex.getMessage());

}
