Сортировка документов

Метод sort()

db.ИМЯ_КОЛЛЕКЦИИ.find().sort( {АТРИБУТ: 1} )

Значение возле атрибута, указывает порядок сортировки:

- для сортировки по возрастанию используется число ‘1’ (по умолчанию документы сортируются по возрастанию)

- для сортировки по убыванию – ‘-1’

Пример - отсортировать документы данной коллекции по имени (по возрастанию):

db.developers.find().pretty().sort( {"title": 1} )
