Projection


Projection (проекция) - необходимые данные документа коллекции.

Метод find()

Чтобы ограничить данные, которые будут получены в результате выполнения запроса нам необходимо указать список атрибутов документа со значениями 1 или 0:

1 – означает, что данный атрибут будет выведен, 0 – нет.

db.ИМЯ_КОЛЛЕКЦИИ.find( {}, {АТТРИБУТ: 1} )

Пример:

db.developers.find().pretty()

{
	"_id" : ObjectId("5815f4e735ee883c37ac8b4d"),
	"title" : "Eugene Suleimanov",
	"specialty" : "Java",
	"skills" : [
		"Java",
		"Hibernate",
		"Spring"
	],
	"salary" : 3500
}
{
	"_id" : ObjectId("5815f4ec35ee883c37ac8b4e"),
	"title" : "Ivan Ivanov",
	"specialty" : "Java",
	"skills" : [
		"Java",
		"Hibernate",
		"Spring"
	],
	"salary" : 3500
}
{
	"_id" : ObjectId("5815f4f535ee883c37ac8b4f"),
	"title" : "Nikolay Nikolaev",
	"specialty" : "C++",

Выводим только специальности разработчиков:

db.developers.find( {}, {_id: 0, "specialty": 1})

{ "specialty" : "Java" }
{ "specialty" : "Java" }
{ "specialty" : "C++" }

Выводим специальности и id документов:

db.developers.find( {}, {"specialty": 1})

{ "_id" : ObjectId("5815f4e735ee883c37ac8b4d"), "specialty" : "Java" }
{ "_id" : ObjectId("5815f4ec35ee883c37ac8b4e"), "specialty" : "Java" }
{ "_id" : ObjectId("5815f4f535ee883c37ac8b4f"), "specialty" : "C++" }
