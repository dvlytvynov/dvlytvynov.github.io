MongoDB Package Content


Содержимое пакета MongoDB C:\Program Files\MongoDB\Server\3.6\bin

• bsondump.exe - считывает содержимое BSON файлов и преобразует их в читабельный формат, например JSON

• mongo.exe - консольный интерфейс для взаимодействия с базами данных

• mongod.exe - сервер базы данных MongoDB, обрабатывает запросы, управляет форматом данных и выполняет различные операции в фоновом режиме по управлению базами данных

• mongodump.exe - утилита создания back-up баз данных

• mongoexport.exe - утилита, позволяет экспортировать данные в формате JSON, TSV или CSV

• mongofiles.exe - утилита, позволяет управлять файлами в системе gridfs для хранения более чем 16 Mб данных

• mongoimport.exe - утилита, импортирующая данные в формате JSON, TSV или CSV в базу данных MongoDB

• mongoperf.exe - утилита, проверяет производительность жесткого диска на предмет операций ввода-вывода

• mongorestore.exe

• mongos.exe - служба маршрутизации MongoDB, которая помогает обрабатывать запросы и определяет местоположение данных в кластере MongoDB

• mongostat.exe

• mongotop.exe - утилита представляет способ подсчета времени, затраченного на операции чтения, записи в базе данных
