Gradle



Gradle пишется на Groovy
Использует репозиторий Maven
Gradle ближе к Ant но с возможностями Maven
Target (у Ant), Goal (у Maven), Task (у Gradle)
Task у Gradle можно менять
Gradle часто вместе с проектом ложит в репозиторий билд-систему (gradlew - sh для Линукса и gradlew.bat файл для Windows. Но с точки зрения репозитория никаких бинарных файлов в репозитроии быть не должно

В корне проекта два файла:
build.gradle - главный файл (написан на Groovy)
settings.gradle

Правой кн на файле build.gradle в IDEA: Link Gradle Project - откроется окно проекта Gradle

В командной строке:
gradle clean
gradle build
gradle clean build
